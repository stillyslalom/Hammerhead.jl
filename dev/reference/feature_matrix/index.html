KA CPU is the hardware-free proving backend selected by backend = :ka. CUDA and AMDGPU accelerate correlation, deformation, peak analysis, and the Wieneke uncertainty statistics; image loading, dewarping, 3C reconstruction, PTV, validation, exports, and statistics remain host operations. GPU backends support cross/phase correlation and gauss3/gauss9 subpixel fits; enlarged search areas, gauss2d, and retained planes are rejected explicitly.

All workflows accept Float32 and Float64. See KernelAbstractions and GPU backends for the precise device boundary and supported options.
